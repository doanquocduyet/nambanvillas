🐙 Tạo tài khoản & Upload code lên GitHub
GitHub là nơi lưu trữ code – bắt buộc để kết nối Vercel
1

Tạo tài khoản GitHub (nếu chưa có)

Vào github.com/signup → Đăng ký bằng email → Xác nhận email → Xong

2

Tạo Repository mới

Sau khi đăng nhập GitHub:

Nhấn nút + New góc trên bên phải
Repository name: nambanvillas
Description: Bất động sản Nam Ban Villas
Chọn: Public
✅ KHÔNG tick "Add README"
→ Nhấn Create repository
3

Upload các file website lên GitHub

Trên trang repo vừa tạo, nhấn "uploading an existing file"

📂 Cấu trúc thư mục cần upload:

4

Kiểm tra lại

Nếu thấy các file đã xuất hiện → ✅ GitHub xong!

⚠️ Lưu ý quan trọng: File .htaccess bắt đầu bằng dấu chấm – trên Windows có thể bị ẩn. Đảm bảo upload đủ file này.

▲ Deploy lên Vercel MIỄN PHÍ
Vercel tự động deploy mỗi khi bạn cập nhật code trên GitHub
1

Đăng ký Vercel bằng GitHub

Vào vercel.com/signup → Chọn "Continue with GitHub" → Cấp quyền cho Vercel → Xong

2

Import dự án từ GitHub

Trên Dashboard Vercel:

Nhấn Add New → Project
→ Chọn tab Import Git Repository
→ Tìm repo nambanvillas
→ Nhấn Import
3

Cấu hình Deploy

Framework Preset: Other
Root Directory: ./ (mặc định)
Build Command: (để trống)
Output Directory: (để trống)
→ Nhấn Deploy
💡 Vì website là HTML thuần nên không cần build command gì cả.
4

Chờ ~1 phút – Website đã live!

Vercel sẽ cấp cho bạn URL dạng: nambanvillas.vercel.app
Vào URL đó kiểm tra website hiện đúng không → ✅ Nếu đúng tiếp tục bước 3

🔄 Khi nào cần cập nhật website?

Chỉ cần upload file mới lên GitHub → Vercel tự động deploy lại trong 1 phút. Không cần làm gì thêm!

🌐 Trỏ tên miền từ PA Vietnam về Vercel
Để website chạy trên nambanvillas.vn thay vì nambanvillas.vercel.app
📋 Bạn cần có: Tên miền đã mua ở PA Vietnam (pa.vn) – ví dụ: nambanvillas.vn hoặc namban.vn

Bước A – Thêm domain vào Vercel

1

Vào Settings dự án trên Vercel

Dashboard → Chọn project nambanvillas
→ Tab Settings
→ Mục Domains
→ Nhập: nambanvillas.vn
→ Nhấn Add
2

Vercel sẽ hiển thị 2 giá trị DNS cần thiết

# Vercel sẽ hiện thông tin dạng này:
Type: A     Name: @     Value: 76.76.21.21
Type: CNAME   Name: www   Value: cname.vercel-dns.com

Ghi lại 2 dòng này, dùng ở bước tiếp theo

Bước B – Trỏ DNS tại PA Vietnam

1

Đăng nhập PA Vietnam

Vào pavietnam.vn → Đăng nhập tài khoản

2

Vào quản lý tên miền

Menu → Tên miền → Quản lý tên miền
→ Tìm tên miền của bạn (vd: nambanvillas.vn)
→ Nhấn Quản lý hoặc biểu tượng ✏️
→ Chọn tab Quản lý DNS hoặc Zone DNS
3

Xóa record cũ (nếu có) và thêm record mới

Xóa hết các record A và CNAME cũ, sau đó thêm:

# Record 1 – trỏ domain chính
Type: A
Host/Name: @ (hoặc để trống)
Value/IP: 76.76.21.21
TTL: 3600
→ Lưu lại

# Record 2 – trỏ www.
Type: CNAME
Host/Name: www
Value: cname.vercel-dns.com
TTL: 3600
→ Lưu lại
⏳ Lưu ý: DNS cần 1–24 giờ để cập nhật toàn cầu (thường 1–2 giờ). Trong thời gian chờ website vẫn truy cập được qua nambanvillas.vercel.app
4

Kiểm tra DNS đã cập nhật chưa

Vào dnschecker.org → Nhập nambanvillas.vn → Kiểm tra
Khi thấy IP 76.76.21.21 xuất hiện → DNS đã cập nhật ✅

5

HTTPS tự động – không cần làm gì

Vercel tự cấp SSL/HTTPS miễn phí cho domain của bạn. Sau 5–10 phút domain active, website sẽ chạy https://nambanvillas.vn ✅

Sau khi deploy – Làm thêm để lên top Google nhanh
1

Đăng ký Google Search Console Miễn phí

Vào search.google.com/search-console → Thêm property → Xác minh quyền sở hữu → Submit sitemap: nambanvillas.vn/sitemap.xml

2

Đăng ký Google Business Profile Miễn phí

Vào business.google.com → Tạo profile cho "Nam Ban Villas" với địa chỉ Nam Ban, Lâm Hà → Giúp hiện trên Google Maps

3

Thêm ảnh thật vào website

Upload ảnh thật của dự án vào thư mục images/ – thay thế ảnh placeholder. Google ưu tiên trang có ảnh chất lượng cao.

4

Điền thông tin liên hệ thật

Mở file index.html, tìm 0xxx xxx xxx → Thay bằng số điện thoại thật. Tương tự với email và Zalo.

5

Đăng bài blog đều đặn

2–3 bài/tuần về thị trường đất Nam Ban, tin tức địa phương → Google sẽ index và đưa website lên cao dần.
